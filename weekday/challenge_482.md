# Challenge 482 - YouTube Upload Count

You are given a list of dates in the format `month day` and a month as arguments. Each date represents a video that was uploaded on that day. Return the number of uploads for the given month.

## Examples
```python
upload_count(["Sept 22", "Sept 21", "Oct 15"], "Sept") ➞ 2

upload_count(["Sept 22", "Sept 21", "Oct 15"], "Oct") ➞ 1
```
